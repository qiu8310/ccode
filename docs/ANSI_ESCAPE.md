## CSI Codes

**结构**

```
CSI [private mode character(s?)] n1 ; n2... [trailing intermediate character(s?)] letter
```


## Terminology

* **CSI** - Control Sequence Introducer (or Control Sequence Initiator)
* **SGR** - Select Graphic Rendition
